核心特性

Curvine 通过创新架构和先进技术提供卓越性能。

极致性能

基于 Rust 构建，提供微秒级延迟和百万级 QPS，优化内存使用和 CPU 效率。

分布式架构

原生分布式设计，支持水平扩展，具备自动故障转移和数据复制功能。

智能内存管理

先进的内存管理策略，支持多级缓存、智能淘汰算法和内存压缩技术。

应用场景

Curvine 适用于各种高性能计算和数据密集型应用场景。

AI/ML 训练

为机器学习训练提供高速数据访问，支持大规模数据集的快速加载和特征缓存。

实时推理

为实时推理服务提供毫秒级响应，支持模型缓存和预测结果缓存。

云上 Agent 原生存储

为 AI Agent 提供可像本地磁盘一样访问的高性能云对象存储，加速数据集、模型产物和上下文数据访问，同时提升可扩展性并降低云存储访问延迟。

大数据计算

为大数据离线计算提供热数据缓存、shuffle加速等，显著提升数据处理效率。

性能指标

Curvine 单集群支持 50 亿小文件，稳定 QPS 10w+，访问时延低至 100us，并以仅 1GB 内存占用实现高效运行。

快速上手

复制一条命令即可启动本地 Curvine Docker 体验集群，验证通过后再进入生产部署。

1

1. 安装 Curvine

使用包管理器或从源码编译安装 Curvine 服务端和客户端。

2

2. 配置集群

配置 Curvine 集群，设置节点发现、数据复制和故障转移策略。

3

3. 开始使用

通过 SDK 或 REST API 连接到 Curvine 集群，开始存储和访问数据。

在终端中运行以下命令
curl -fsSL https://curvineio.github.io/install.sh | bash

如果你的机器已经安装并可访问 Docker，请在终端运行以下命令，然后根据屏幕提示启动本地 Curvine 体验集群。
